# code-demo-css3-animation

# HTML Files
-start.html: basic html code for header and navbar, instructions in the comments throughout the document.
-finished.html: animated header and navbar. this is what it should be like after completing the code demo.

# CSS Files
-css/startstyles.css: basic css for header and navbar style, animation code needed from 'copystyles.css'.
-css/copystyles.css: animation code snippets that will add animation, copy/paste into 'startstyles.css'.
-css/styles.css: style sheet for finished.html to see what the animations should look like at the end.
